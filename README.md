# Calculator Light/Dark Theme

A responsive calculator with a neumorphic light and dark theme toggle.

# Link  

https://suman-max10.github.io/Calculator-LightDark-Theme/

## Features

- Basic arithmetic operations: add, subtract, multiply, divide
- Decimal input and percentage support
- Clear and sign toggle controls
- Light and dark theme switch
- Pure HTML, CSS, and JavaScript

## Files

- [index.html](index.html)
- [style.css](style.css)
- [script.js](script.js)

## How to Run

1. Open `index.html` in a web browser.
2. Use the calculator buttons to enter expressions.
3. Toggle the switch at the top right to change themes.

## Notes

- The calculator uses `eval()` for expression evaluation.
- The layout is built with CSS Grid and neumorphic styling.
